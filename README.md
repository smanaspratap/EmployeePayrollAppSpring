# EmployeePayrollAppSpring

A **Spring Boot REST Application** built as part of the BridgeLabz Full-Stack Backend Training.  
This app serves as the backend for the Employee Payroll UI, replacing the JSON Server with a  
fully functional Spring Boot REST API backed by an **H2 in-memory database** and later **MySQL**.
